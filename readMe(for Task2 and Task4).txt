Приложение Task2(папка Task2 - копия) представляет собой консольное приложение обеспечивающее ввод с клавиатуры 
набора цифр и сортирующее его (набор) по возрастанию. На старте программы требуется ввести параметр "N" - количество 
символов для будущей сортировки, и нажать ВВОД. Далее введите через пробел сами цифры (их количество должно быть 
равно параметру N) и снова ВВОД.
